# ITCSS
A movie search app using React, Redux and ITCSS (Inverted Triangle CSS).

## Installation

1. `git clone http://github.com/sani723/ITCSS`
2. To run client `npm start`

and go to `http://localhost:3000/`

## Flow

```
flow check
or
npm run flow
```

## Built with

* React (v16)
* Flow
* Redux
